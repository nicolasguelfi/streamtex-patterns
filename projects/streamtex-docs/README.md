# Project-specific patterns — streamtex-docs

Patterns ultra-specific to the `streamtex-docs` manuals that don't fit
into `core/`, `slides/` or `docs/`.

Currently: empty. The bulk of streamtex-docs needs is covered by
patterns in `docs/` (`manual_section`, `feature_walkthrough`,
`api_reference_card`, `composite_block`).

Add patterns here only if a manual needs a layout that no other
StreamTeX project would reuse.
